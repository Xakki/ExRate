# Анализ FRED API (Federal Reserve Economic Data)

## 1. Обзор доступных данных

FRED API предоставляет доступ к огромному количеству экономических временных рядов (более 800 000), собранных из различных источников. Все запросы требуют API-ключ.

### Группировка данных и эндпоинты

Данные в FRED структурированы по следующим категориям:

#### А. Категории (Categories)
Используются для иерархической навигации по типам данных (например, "Денежные показатели", "Цены", "Производство").
- **Пример запроса**: `https://api.stlouisfed.org/fred/category?category_id=125&api_key={key}&file_type=json`
- **Зачем**: Позволяет найти нужные ряды данных, двигаясь от общих тем к частным.

#### Б. Выпуски (Releases)
Данные группируются по статистическим публикациям (например, "Consumer Price Index" или "H.10 Foreign Exchange Rates").
- **Пример запроса**: `https://api.stlouisfed.org/fred/releases?api_key={key}&file_type=json`
- **Зачем**: Позволяет отслеживать новые поступления данных и понимать их происхождение.

#### В. Серии (Series)
Это основные временные ряды. Содержат метаданные (название, частота, единицы измерения) и сами наблюдения (значения).
- **Пример запроса (Метаданные)**: `https://api.stlouisfed.org/fred/series?series_id=DEXUSEU&api_key={key}&file_type=json`
- **Пример запроса (Значения)**: `https://api.stlouisfed.org/fred/series/observations?series_id=DEXUSEU&api_key={key}&file_type=json`
- **Зачем**: Основной способ получения конкретных цифр (курсов валют, ВВП и т.д.).

#### Г. Источники (Sources)
Информация об организациях, предоставляющих данные (например, Бюро трудовой статистики США, Европейский центральный банк).
- **Пример запроса**: `https://api.stlouisfed.org/fred/source?source_id=1&api_key={key}&file_type=json`

#### Д. Теги (Tags)
Позволяют фильтровать данные по атрибутам (регион, единица измерения, частота).
- **Пример запроса**: `https://api.stlouisfed.org/fred/tags?api_key={key}&file_type=json`

## 2. Валютные курсы (Foreign Exchange Rates)

`<release id="17" realtime_start="2013-08-13" realtime_end="2013-08-13" name="H.10 Foreign Exchange Rates" press_release="true" link="http://www.federalreserve.gov/releases/h10/"/>`

`https://api.stlouisfed.org/fred/release/series?release_id=17&api_key=***`

Данные по валютам в FRED в основном поступают из релиза **H.10 Foreign Exchange Rates** Совета управляющих Федеральной резервной системы.

### Доступные валютные пары (примеры)
В основном котировки идут по отношению к доллару США (USD).
- **DEXUSEU**: USD за 1 Евро
- **DEXJPUS**: Японская иена за 1 USD
- **DEXCHUS**: Китайский юань за 1 USD
- **DEXCAUS**: Канадский доллар за 1 USD
- **DEXUSUK**: USD за 1 Фунт стерлингов
- **DEXMXUS**: Мексиканское песо за 1 USD
- **DEXBZUS**: Бразильский реал за 1 USD
- **DEXSZUS**: Швейцарский франк за 1 USD
- **DEXINUS**: Индийская рупия за 1 USD
- **DEXKOUS**: Южнокорейская вона за 1 USD

### Параметры выборки
- **Частота**: Ежедневно (Daily), за исключением выходных и праздничных дней.
- **Период выборки**: Для большинства основных валют история доступна с **1971 года**. Для евро (DEXUSEU) — с января 1999 года.
- **Задержка данных (Lag)**: Данные обновляются **еженедельно** (обычно по понедельникам). В релиз включаются данные за предыдущую рабочую неделю. Таким образом, актуальная задержка составляет от **4 до 11 дней** в зависимости от дня запроса.

## 3. Лимиты и ограничения

### Технические лимиты
- **Частота запросов**: Официально лимит может меняться, но на практике составляет около **120 запросов в минуту**.
- При превышении лимита API возвращает ошибку `429 Too Many Requests`.

### Условия использования
- **Запрет на кэширование**: Условия использования запрещают создание полных архивов или баз данных на основе FRED. Кэширование допускается только для временных нужд приложения.
- **AI/ML**: Запрещено использовать данные FRED для обучения LLM или других моделей ИИ без специального разрешения.
- **Авторские права**: Некоторые данные в FRED принадлежат сторонним организациям (например, индексы S&P) и могут иметь дополнительные ограничения.

## 4. Пример получения курса EUR/USD

1. **Поиск серии**: `https://api.stlouisfed.org/fred/series/search?search_text=exchange+rate+euro&api_key=****&file_type=json`
2. **Получение данных за период**:
```bash
curl "https://api.stlouisfed.org/fred/series/observations?series_id=DEXUSEU&observation_start=2026-01-01&api_key=****&file_type=json"
```
